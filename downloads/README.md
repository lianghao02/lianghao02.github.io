# Photo Report 自動化工具使用說明

## 簡介

這是一個專為警務工作設計的 Excel VBA 自動化工具，能夠快速將照片整理並套印至指定的表格格式中，大幅節省人工排版的時間。

## 功能特色

- **自動匯入**: 批量匯入資料夾內的照片。
- **自動排版**: 依照預設格式自動調整照片大小與位置。
- **快速輸出**: 支援直接列印或另存為 PDF。

## 系統需求

- 作業系統: Windows 10/11
- 軟體需求: Microsoft Excel (需啟用巨集功能)

## 使用步驟

### 1. 準備照片

- 將需要製作報告的照片整理在同一個資料夾中。
- **注意**: 本工具僅支援 **JPG** 或 **PNG** 格式。
- 若您的照片為 **HEIC** 格式 (iPhone 預設)，請先使用我們提供的 [Police Image Converter](https://lianghao02.github.io/police-image-converter/) 進行轉檔。

### 2. 開啟工具

- 點擊 `Photo_Report.xlsm` 開啟檔案。
- 若 Excel 上方出現「安全性警告」，請點擊 **[啟用內容]** 以允許巨集執行。

### 3. 執行匯入

- 點擊工具介面上的 **[匯入照片]** 按鈕。
- 選擇您存放照片的資料夾。
- 系統將自動讀取並排版。

### 4. 輸出報告

- 確認排版無誤後，可直接列印或儲存檔案。

## 常見問題

**Q: 為什麼照片無法匯入？**
A: 請確認照片檔名是否包含特殊字元，或格式是否為 JPG/PNG。

**Q: 執行時出現錯誤訊息？**
A: 請確認您已啟用 Excel 的巨集權限。
